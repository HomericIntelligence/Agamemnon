## What Worked

- **Hierarchical closure-based action maps** — Pre-seed mutable state before building action maps if resuming; use mutable namespace dicts for multi-level shared state
- **Sentinel exceptions for coordinated shutdown** — Define sentinels in the module owning the shutdown flag to avoid circular imports; catch BEFORE `except Exception` in all hierarchical state machine levels
- **Atomic checkpoint writes + temp file + rename pattern** — Always use atomic rename to prevent corrupt JSON if killed mid-write; enables safe recovery even on process crash
- **Incremental checkpoint saves** — Save checkpoint after each item completes (not just at end); enables granular recovery despite interruptions
- **Dual persistence (local + remote)** — Save to checkpoint file first (source of truth during runtime), then sync to backing store (GitHub Issues) after state transitions
- **Config hash validation** — Exclude testing-only params from hash; strict validation prevents silent config drift on resume
- **Completion vs. Success distinction** — Treat ALL results (pass, fail, error) as "completed" by default; provide selective re-run flag for user control
- **Workspace idempotency checks** — Check if work is already done before executing; prevents duplicate state on resume
- **Intermediate state detection** — Scan for non-terminal states in "complete" experiments; reset affected tiers to re-enter state machine from checkpoint
- **NATS direct connection over Tailscale** — For 2-host topologies, direct connection simpler than leaf nodes; Tailscale handles routing

## What Failed

| Attempt | Why It Failed | Fix |
|---------|---------------|-----|
| Only saving checkpoint at end of batch | Ctrl+C or crash loses ALL progress; long jobs restart from scratch | Save checkpoint after EACH item completes |
| Writing checkpoint directly (no temp file) | Process killed mid-write corrupts JSON; recovery fails | Use temp file + atomic rename pattern |
| Define sentinels in state_machine.py | Circular import: state_machine imports runner, runner imports state_machine | Define sentinels in module owning shutdown flag (e.g., runner.py) |
| Treat errors as incomplete (auto-retry) | Permanent errors waste time on re-run (bad fixture, missing deps) | Treat ALL results completed by default; user controls retry via flag |
| Catch KeyboardInterrupt in state machine | subprocess.run() returns normally with signal exit code (no exception) | Check is_shutdown_requested() explicitly AFTER subprocess returns |
| Reset tier to SUBTESTS_RUNNING on interrupt | SUBTESTS_RUNNING expects aggregated results already present | Reset to CONFIG_LOADED which re-enters subtest execution from checkpoint |
| Let ShutdownInterruptedError propagate through ProcessPoolExecutor wrapper | Safe wrapper's `except Exception` converts to WorkerError, swallowing signal | Explicitly check `if isinstance(e, ShutdownInterruptedError): raise` |
| Check pre-advance state for --until condition | `current = get_state()` then `advance()` fires too late — action already ran | Capture state AFTER advance: `new_state = advance()` then check |
| Call terminal_guard() with no arguments | shutdown_fn defaults to None; signal handlers never installed | Pass shutdown function: `terminal_guard(request_shutdown)` |
| Connect NATS leaf.conf to port 4222 | Leaf nodes require dedicated leafnode listener port (7422) | Always use port 7422 in leaf.conf |
| Run NATS in podman with slirp4netns | Container port remapping makes NATS invisible to external hosts | Start NATS as native binary when rootlessport absent |
| Load completed runs from report.json | report.json has simplified structure; no from_dict() classmethod | Create separate run_result.json with full data; deserialize on resume |
| Monolithic reset logic in one function | C901 complexity > 10; hard to test and maintain | Extract each concern into separate method; use shared state constants |

```python
def save_task_brief_checkpoint(brief: TaskBrief, brief_id: str, checkpoint_dir: Path):
    checkpoint_path = checkpoint_dir / f"{brief_id}.json"
    tmp_path = checkpoint_path.with_suffix(".json.tmp")
    
    checkpoint = {
        "version": "1.0",
        "brief_id": brief_id,
        "original_task_brief": brief.to_dict(),
        "completed_at": brief.completed_at.isoformat() if brief.completed_at else None,
        "github_issue_synced": False,
        "last_updated_at": datetime.utcnow().isoformat(),
    }
    
    with open(tmp_path, "w") as f:
        json.dump(checkpoint, f, indent=2)
    tmp_path.rename(checkpoint_path)  # Atomic on POSIX
```

```python
def compute_brief_config_hash(brief: TaskBrief) -> str:
    hash_input = {
        "id": brief.id,
        "research_question": brief.research_question,
        "decomposition_strategy": brief.decomposition_strategy,
        "hmas_layers": [layer.value for layer in brief.hmas_layers],
        "deadline": brief.deadline.isoformat() if brief.deadline else None,
    }
    return hashlib.sha256(
        json.dumps(hash_input, sort_keys=True).encode()
    ).hexdigest()[:16]
```
